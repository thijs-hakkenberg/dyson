# Control of a Bucket-Wheel for Surface Mining of Asteroids

## Abstract

This paper investigates excavation mechanisms for Near Earth Asteroids using dual counter-rotating bucket-wheels on a hovering spacecraft. The system extracts regolith, heats it to 1000°C to liberate water, then electrolyzes it for hydrogen and oxygen fuel production.

## Key Control Challenges

1. **Traction maximization**: Counter-rotating wheels balance torques
2. **Lift-off prevention**: Reaction forces in microgravity
3. **Hovering stability**: Maintaining position during excavation
4. **Regolith flow control**: Managing material into heating chamber

## Technical Parameters

| Parameter | Value |
|-----------|-------|
| Heating temperature | 1000°C |
| Configuration | Dual counter-rotating wheels |
| Output | H₂ + O₂ propellant |
| Application | In-situ fuel production |

## Relevance to Project Dyson Phase 0

Directly applicable to mining robot design:

1. **Excavation mechanism**: Bucket-wheel is proven for low-g
2. **Torque balancing**: Counter-rotation solves reaction force problem
3. **Integrated processing**: Excavation + heating + electrolysis in one unit
4. **Propellant production**: Enables refueling for extended operations
